Modo inmersivo

Modo navegador

INSTRUCCIONES


1. Pulse la esfera para iniciar la animación de los paquetes


2. Puede pulsar indefinidamente la esfera para

pausar y reaunudar el movimiento de los paquetes


3. Pulse en los nodos para desplegar la tabla

de encaminamiento


4. Los mensajes muestran la información de cada protocolo

según el color, pulsando en cada nivel se modifica lo mostrado:

  • Amarillo: Ethernet

  • Naranja: ARP

  • Verde: IP

  • Rojo: ICMP

  • Azul: TCP

  • Blanco: Datos sobre TCP o UDP

  • Rosa: UDP

  • Oro viejo: DNS

  • Oro: HTTP

4. Pulse en cualquier nivel para ver su información desplegada
