VRNetVis: demos DNS

En pc1 se ejecuta host pc2.emp2.net (consulta de DNS). El servidor DNS de pc1 es dnsemp1. dnsroot1 y dnsroot2 son servidores del dominio raíz. dnsnet es servidor del dominio .net. dnsemp2.net es servidor del dominio emp2.net.

  • Sólo nivel de aplicación: mensajes DNS. La primera consulta es recursiva, el resto iterativas.
  • Nivel de aplicación, transporte (UDP) y red.
